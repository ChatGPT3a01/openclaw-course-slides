0.1 什麼是自主代理 AI？

你一定用過 ChatGPT 或 Gemini 吧？打開網頁、輸入問題、AI 回答你。但它有一個根本性限制：它永遠在等你。你不問，它就不動。

自主代理 AI 不一樣——它像一個住在你家的私人管家。你交代一句「幫我處理今天的事」，等你回來，所有事情都搞定了。

類型比喻你需要做的事
聊天機器人📖 百科全書每次翻開查一個問題
AI 助手（Siri）🎙️ 語音遙控器每次說一個指令
自主代理 AI🦞 私人管家交代一次，它自己安排好
💡 一句話總結：聊天機器人是「你問它答」，自主代理是「你交代，它自己去完成」。

0.2 Agent 世代演進

🔧 第一代：Coding Agent

2023 年｜GitHub Copilot、Cursor

只會在編輯器裡寫程式碼。像一個打字超快的助理——但只會打字，不會跑腿。

🌐 第二代：Browser Agent

2024 年｜Browser Use、Playwright AI

能操作瀏覽器、填表單、擷取資料。AI 終於從編輯器裡「走出來」了。

🖥️ 第三代：Desktop Agent

2025 年｜Anthropic Computer Use

能操作整個桌面！但只限面前這台電腦，關機就停擺。

🦞 第四代：Computer Agent

2026 年｜OpenClaw

整合前三代 + 多通道通訊 + 24hr 運行 + 多機協作 + 技能擴充 + 語音通話！

🦞 OpenClaw 龍蝦 AI — 你的 AI 管家

一個永遠在線的 AI 管家，24 小時不間斷等待你的指令。由奧地利開發者 Peter Steinberger 創建，2026 年 1 月問世，完全免費開源。

項目說明
費用開源免費（搭配 AI 模型：Gemini 免費 / OpenAI OAuth 免費 / 付費 API）
支援平台Windows / macOS / Linux / 雲端
通訊管道LINE、Telegram、WhatsApp、Discord 等 22 個
官方技能70+ 個（天氣、Email、日曆、自拍、語音通話等）
💡 為什麼叫龍蝦？OpenClaw 的「Claw」是爪子的意思。龍蝦最大的特徵就是那對大爪子 🦞
什麼時候要開 OpenClaw？
當你要在 LINE / Telegram 直接跟 AI 對話、查資料、收通知、叫它幫你做生活任務時，就開它。
它不是拿來寫程式的主工具，它是你最後真的要「每天用」的那隻 AI。

🚀 Google Antigravity — 你的 AI 工程師

Google 推出的 AI 開發環境。長得像 VS Code，但你不需要會寫程式——用中文描述你想要什麼，它就自動幫你生成完整的應用程式。

項目說明
費用個人免費（公開預覽版）
AI 模型Gemini 3.1 Pro / Gemini 3 Flash
特色Agent Manager 多代理協作、內建瀏覽器

📝 Editor View

即時寫程式、補全、除錯。適合你想一邊看一邊改。

👥 Manager View

同時派出多個 AI Agent 分頭工作，互不干擾。

什麼時候要開 Antigravity？
當你想快速生出一個頁面、一個小工具、一個專案骨架，或要讓 AI 分工處理多件開發工作時，用它最快。
它比較像「做東西」的地方，不是平常拿來聊天的地方。
