22.1 從「看懂」到「看清楚」

CH8 教過用 AI 視覺模型看圖，但那只是「描述照片」。YOLO6 則是精準框出每一個物件，告訴你「那是什麼」和「有幾個」。

比較項目CH8 AI 視覺YOLO6 物件偵測
做的事描述照片內容框出每個物件 + 分類 + 計數
回答方式「照片裡有幾個人在聊天」「偵測到 3 人、2 張椅子、1 張桌子」
速度需要呼叫雲端 API（1-3 秒）本地即時偵測（毫秒級）
精準度模糊描述座標 + 信心分數
離線使用需要網路完全離線

🗣️ CH8：AI 看圖說話

「這是一張教室的照片，裡面有一些學生在上課。」

👁️ YOLO6：精準偵測

「person: 28 個、chair: 30 張、laptop: 5 台、book: 12 本」

💡 最佳搭配：YOLO6 負責「偵測 + 計數」，再把結果丟給 AI 模型「解讀 + 回應」——兩者結合才是完整的智慧視覺！

模型大小選擇

模型大小速度精準度適用場景
yolo11n~6 MB最快普通樹莓派、手機

22.3 第一次偵測

完整 Python 程式碼（只要 10 行）

from ultralytics import YOLO

# 1. 載入模型（第一次會自動下載）
model = YOLO("yolo11s.pt")

# 2. 偵測圖片
results = model("classroom.jpg")

# 3. 顯示結果
results[0].show()       # 彈出標註過的圖片
results[0].save()       # 儲存標註圖片
print(results[0].boxes) # 印出偵測框座標

22.4 做成龍蝦 Skill

把 YOLO6 包裝成龍蝦技能，用聊天就能觸發偵測！

SKILL.md 核心內容

# yolo-vision

## 工具名稱
yolo-vision

## 描述
使用 YOLO6 進行即時物件偵測，能精準框出照片中的每個物件並計數。

## 功能
- 接收使用者傳送的照片
- 使用 YOLO6 模型偵測物件
- 回傳偵測結果（物件類別 + 數量 + 位置）
- 支援 80 種常見物件

## 使用方式
傳送照片並說「幫我分析」或「偵測這張照片」即可觸發。

安裝三步驟

  1. 建立資料夾：mkdir ~/.openclaw/workspace/skills/yolo-vision
  2. 寫入檔案：把 SKILL.md 和 detect.py 放進去
  3. 重啟龍蝦：openclaw gateway restart
✅ 安裝完成後，傳張照片給龍蝦，說「幫我分析這張照片有什麼東西」——它會呼叫 YOLO6 偵測後，用自然語言告訴你結果！

即時偵測 Python 程式碼（精簡版）

from ultralytics import YOLO
import cv2

model = YOLO("yolo11s.pt")
cap = cv2.VideoCapture(0)  # 開啟 Webcam

while True:
    ret, frame = cap.read()
    if not ret: break
    results = model(frame, verbose=False)
    annotated = results[0].plot()    # 畫上偵測框
    cv2.imshow("YOLO6 Live", annotated)
    if cv2.waitKey(1) == ord('q'): break

cap.release()
cv2.destroyAllWindows()

22.9 常見問題

  • Q：需要 GPU 嗎？
    A：不需要！yolo11n 和 yolo11s 在 CPU 上就能跑。有 GPU 會更快，但不是必要條件。即使是 5 年前的筆電也能做定時截圖偵測。
  • Q：YOLO6 能偵測什麼？
    A：預設支援 80 種常見物件，包括：人、車、自行車、貓、狗、椅子、桌子、筆電、手機、書本、瓶子、杯子、背包等。完整清單可用 model.names 查看。
  • Q：能和 CH8 的 AI 視覺同時使用嗎？
    A：當然可以！最佳搭配就是 YOLO6 先「偵測 + 計數」，再把結果和照片一起丟給 AI 模型做「語意理解 + 自然語言回應」。兩者互補，不衝突。
  • Q：偵測不準怎麼辦？
    A：三個方向調整——換大一號的模型（n → s → m）、調整信心閾值（預設 0.25，改高一點過濾雜訊）、確保照片光線充足且角度正面。
  • Q：手機上能跑嗎？
    A：YOLO 模型可以匯出成手機格式（CoreML / TFLite），但龍蝦技能目前是跑在電腦上。最簡單的做法是：手機拍照 → 傳給龍蝦（LINE） → 電腦端用 YOLO 偵測 → 結果回傳手機。
