22.3 步驟二（2/3）：寫偵測程式

新建一個檔案 detect_test.py，只要 10 行就能完成偵測：

from ultralytics import YOLO      # 1. 載入 YOLO 模組

model = YOLO("yolo11s.pt")        # 2. 載入模型（第一次會自動下載 ~20MB）

results = model("test.jpg")       # 3. 偵測你的照片

results[0].show()                  # 4. 彈出標註過的圖片（彩色框框）
results[0].save()                  # 5. 儲存標註圖片到 runs/ 資料夾

# 6. 列出每個偵測到的物件
for box in results[0].boxes:
    cls_name = model.names[int(box.cls)]
    confidence = float(box.conf) * 100
    print(f"偵測到：{cls_name}（信心度 {confidence:.1f}%）")

執行偵測

python detect_test.py
💡 第一次執行時，YOLO 會自動下載模型檔（yolo11s.pt，約 20MB），之後就不用重複下載了。

22.4 步驟三（4/5）：寫偵測腳本

建立 detect.py，這個腳本做三件事：載入模型 → 偵測照片 → 輸出 JSON 結果。

import sys, json
from ultralytics import YOLO
from collections import Counter

# 1. 載入模型
model = YOLO("yolo11s.pt")

# 2. 偵測照片（從命令列參數接收路徑）
image_path = sys.argv[1] if len(sys.argv) > 1 else "test.jpg"
results = model(image_path, verbose=False)

# 3. 統計物件數量
detections = []
for box in results[0].boxes:
    detections.append(model.names[int(box.cls)])

counts = dict(Counter(detections))

# 4. 輸出 JSON（龍蝦讀取這個結果）
output = {
    "total_objects": len(detections),
    "counts": counts,
    "image": image_path
}
print(json.dumps(output, ensure_ascii=False, indent=2))

執行測試

python detect.py test.jpg
# 輸出：{"total_objects": 75, "counts": {"person": 28, "chair": 35, "laptop": 12}}
✅ 程式會輸出 JSON，龍蝦就能讀懂並轉成自然語言回覆給你。

22.4 步驟三（5/5）：寫監控腳本 + 啟動！

建立 webcam_monitor.py，讓龍蝦能用 Webcam 即時監控：

import sys, json, time
from ultralytics import YOLO
from collections import Counter
import cv2

model = YOLO("yolo11s.pt")

# 開啟 Webcam 拍一張快照
cap = cv2.VideoCapture(0)
time.sleep(1)  # 等待攝影機預熱
ret, frame = cap.read()
cap.release()

if not ret:
    print(json.dumps({"error": "無法開啟攝影機"}))
    sys.exit(1)

# 偵測這一幀
results = model(frame, verbose=False)

# 統計
detections = []
for box in results[0].boxes:
    detections.append(model.names[int(box.cls)])

counts = dict(Counter(detections))

# 儲存標註圖片
annotated = results[0].plot()
save_path = "webcam_snapshot.jpg"
cv2.imwrite(save_path, annotated)

output = {
    "total_objects": len(detections),
    "counts": counts,
    "snapshot": save_path
}
print(json.dumps(output, ensure_ascii=False, indent=2))

openclaw gateway restart

22.7 進階：即時偵測

不只分析照片——接上 Webcam，讓畫面上每一幀都即時偵測，物件被框出來的效果非常震撼。

完整程式碼：live_detect.py（15 行）

from ultralytics import YOLO
import cv2

model = YOLO("yolo11s.pt")        # 載入模型
cap = cv2.VideoCapture(0)          # 開啟 Webcam

while True:
    ret, frame = cap.read()
    if not ret: break

    results = model(frame, verbose=False)
    annotated = results[0].plot()  # 畫上偵測框
    cv2.imshow("YOLO6 Live", annotated)

    if cv2.waitKey(1) == ord('q'): # 按 Q 退出
        break

cap.release()
cv2.destroyAllWindows()

22.8 常見問題

  • Q1：一定需要 GPU 嗎？
    A：不需要！yolo11n 在 CPU 上就能跑，5 年前的筆電也行。GPU 只是讓速度更快（從 2 秒變 0.1 秒），但偵測結果是一樣的。
  • Q2：YOLO6 能偵測什麼東西？
    A：預設 80 種常見物件（人、車、動物、家具、3C、食物等）。用 model.names 可以查看完整清單。如果需要偵測清單上沒有的東西，可以自訂訓練模型。
  • Q3：能和 CH8 的 Vision 同時用嗎？
    A：這是最佳搭配！YOLO6 先「偵測 + 計數」提供精確數據，AI Vision 再用自然語言整理成人類看得懂的回覆。兩者互補。
  • Q4：偵測不準怎麼辦？
    A：三個方法——換大一號模型（n → s → m）、確保拍照光線充足、物件不要太小或被嚴重遮擋。
  • Q5：能在手機上跑 YOLO6 嗎？
    A：目前本書範例都在電腦上跑。但 YOLO 有 ONNX / CoreML / TFLite 匯出格式，理論上可以在手機執行——這是進階主題。
