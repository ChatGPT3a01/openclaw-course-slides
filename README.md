## 使用方式

1. 用瀏覽器開啟 `index.html` 進入統整頁
2. 選擇篇章進入分類頁，再選擇章節
3. 章節簡報支援：
   - **左右箭頭**或**鍵盤方向鍵**翻頁
   - **底部進度點**跳轉任意頁
   - **F 鍵**全螢幕模式
