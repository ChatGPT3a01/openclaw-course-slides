3.1 Claude Code 是什麼？

Claude Code 是 Anthropic 公司推出的終端機 AI 助手。所謂「終端機」，就是那個讓你輸入指令、按 Enter 執行的命令列視窗（Windows 上叫做 PowerShell）。

一切都是文字——你打字問它，它打字回你，中間穿插著它在你電腦上執行的各種操作。正因為直接在終端機裡運作，它可以執行任何你在命令列上能做的事情。

Claude Code 擅長什麼？

場景用 Antigravity用 Claude Code
從零做一個網頁✅ 更適合（有預覽）可以做但看不到即時預覽
修改一個設定檔可以但大材小用✅ 更適合（快速精準）
批次重命名 100 個檔案不太方便✅ 一個指令搞定
Git 操作（commit、push）可以但步驟多✅ 用對話就好
安裝和設定軟體可以✅ 直接執行指令
理解整個專案結構可以✅ 快速掃描全部檔案
💡 簡單的選擇原則：要「建造」新東西，用 Antigravity；要「操作」現有的東西，用 Claude Code。
2 / 10

3.2 安裝前置條件：Node.js

Claude Code 需要 Node.js 才能運行。安裝過程非常簡單，只需要幾個步驟。

【步驟 1】前往 Node.js 官網

打開瀏覽器，前往 https://nodejs.org/，選擇左邊的 LTS（長期支援版），點擊下載。

【步驟 2】執行安裝程式

下載完成後，執行安裝程式。一路按「Next」、「I agree」、「Install」就好，不需要更改任何設定。

【步驟 3】驗證安裝

打開 PowerShell（按 Windows 鍵，搜尋 PowerShell），輸入：

node --version

如果看到類似 v22.x.x 或 v24.x.x 的版本號碼，就表示安裝成功了。

💡 已經裝過 Node.js 了？只要版本是 v18 以上就可以。輸入 node --version 檢查一下就好。
3 / 10

3.2 安裝 Claude Code 與首次啟動

【步驟 1】安裝 Claude Code

Node.js 裝好之後，在 PowerShell 裡輸入：

npm install -g @anthropic-ai/claude-code

安裝過程可能需要一到兩分鐘。安裝完成後，確認版本：

claude --version

【步驟 2】首次啟動與登入

在 PowerShell 裡輸入 claude，第一次啟動會要求登入：

  • Anthropic API Key：把 sk-ant- 開頭的 Key 貼上去
  • Anthropic 帳號登入：透過瀏覽器授權
  • 訂閱方案：Claude Pro / Max 訂閱
✅ 登入之後，你會看到一個簡潔的對話介面——一個輸入游標在等你打字。這就是 Claude Code 的全部介面了！
4 / 10

3.3 基本對話操作

Claude Code 跟一般聊天機器人最大的不同：它不只是回答問題，它會直接在你的電腦上行動。

練習一：打個招呼

你好，請用中文跟我說話

Claude Code 會用中文回覆你。跟 ChatGPT 一樣，只是發生在終端機裡。

練習二：問它系統問題

我的電腦上有裝 Node.js 嗎？是什麼版本？

它會真的去你的電腦上執行 node --version，然後把結果告訴你。

練習三：探索資料夾

幫我看一下目前這個資料夾裡有什麼

Claude Code 會列出所有的檔案和子資料夾，通常還會附上簡短說明。

⚡ Claude Code 的工作目錄：啟動時會以你在 PowerShell 裡所在的目錄作為起點。想操作特定資料夾，可以先用 cd D:\Moltbot 切換過去，再輸入 claude。
5 / 10

3.4 檔案操作

讀取、修改、建立、搜尋——全部用對話就能完成，不需要自己打開檔案、找到那一行、小心翼翼地修改。

📖 讀取檔案

幫我讀取 config.json 的內容

會把檔案完整內容顯示出來，還附上它的理解和說明。

✏️ 修改檔案

幫我把 config.json 裡的 port
從 3000 改成 18789

先讀取、找到、修改、存檔，兩秒內完成。還會告訴你改了什麼。

📄 建立新檔案

幫我建立一個 hello.txt，
內容寫「第一個檔案」

Claude Code 會建好檔案，你可以用檔案總管打開確認。

🔍 搜尋內容

幫我在這個專案裡搜尋
所有包含 "API_KEY" 的檔案

快速掃描所有檔案，列出哪些檔案的哪幾行包含關鍵字。

⚠️ 安全機制：Claude Code 在修改檔案或執行可能影響系統的指令之前，通常會先跟你確認「是否允許這個操作？」你可以選擇允許或拒絕。
6 / 10

3.5 Git 版本控制——你的檔案時光機

Git 就是你檔案的「時光機」。它會幫你記住每一次修改的歷史——你什麼時候改了什麼、為什麼改。有了 Claude Code，你完全不需要學 Git 的指令。

操作跟 Claude Code 說什麼背後的 Git 指令
初始化 Git幫我初始化 Gitgit init
儲存一個版本幫我 commit，訊息寫「完成首頁」git add + git commit
查看歷史幫我看最近的 Git 歷史紀錄git log
回到舊版本幫我把 config.json 恢復到上一版git checkout / restore
💡 Commit 訊息的意義：每次 commit 都附上簡短說明，記錄這次你做了什麼改動。好的 commit 訊息像日記一樣，讓你的「時光旅行」更有效率。
✅ 有了 Git，你永遠不用擔心「改壞了回不去」。如果改壞了什麼，跟 Claude Code 說一聲就能復原！
7 / 10

3.6 Voice Mode 與斜線指令

🎤 Voice Mode——用說的跟 AI 對話

按住鍵盤上的空白鍵（Space）不放，就會進入語音模式。對著麥克風說出你的需求（中文即可），說完後放開空白鍵。Claude Code 會把語音轉成文字，然後像平常一樣執行操作。

🙌 手邊在忙

看著另一份文件時，同時讓 Claude Code 做事

📝 長篇描述

複雜需求用說的比打字快

♿ 行動不便

暫時不方便打字時使用

⌨️ 斜線指令（Slash Commands）

指令功能什麼時候用
/help顯示說明文件不確定怎麼操作的時候
/clear清除對話歷史想重新開始的時候
/compact壓縮對話記憶對話太長、反應變慢時
/cost顯示 API 使用量想確認花了多少錢時
/doctor診斷設定遇到問題的時候

3.9 VS Code 整合與實戰練習

在 VS Code 裡使用 Claude Code

  1. 打開 VS Code
  2. 按 Ctrl+` 打開終端機面板
  3. 在終端機裡輸入 claude

一邊在 VS Code 的編輯區看檔案，一邊在下方的終端機裡跟 Claude Code 對話。修改的檔案會即時反映在編輯區。
