20.1 Twilio 是什麼？

Twilio 是全球最大的「通訊平台即服務」（CPaaS）公司。它讓開發者透過 API 來打電話、傳簡訊。你不需要了解電話系統底層技術——Twilio 都幫你包好了。

Twilio 能做什麼

功能說明
接聽電話給龍蝦一個電話號碼，使用者打來龍蝦自動接聽
撥打電話龍蝦主動撥打電話給指定號碼
語音辨識來電者說的話，即時轉成文字給龍蝦
語音合成龍蝦的文字回覆，用語音唸出來給來電者聽
簡訊龍蝦也可以收發 SMS 簡訊

20.1.3 Twilio 費用說明

Twilio 不是免費的——它是商業服務。但費用比你想像的低：

項目費用（美元）說明
電話號碼$1.15/月美國號碼月租費
接聽電話$0.0085/分鐘每分鐘不到 0.3 台幣
語音辨識$0.02/15 秒Twilio 內建的 STT
語音合成免費~$0.01/句看你用哪種 TTS
撥打電話$0.014/分鐘撥打美國號碼
✅ 免費試用：Twilio 新帳號有免費額度（約 $15），足夠你測試和學習。你不需要一開始就花錢。
⚠️ 台灣號碼也有，但月租費較高（約 $11/月）。如果只是學習和測試，用美國號碼就夠了。

20.2 申請 Twilio 帳號

註冊步驟

  1. 到 https://www.twilio.com/try-twilio 註冊
  2. 填寫 Email、密碼
  3. 驗證手機號碼（台灣的 +886 號碼可以用）
  4. 回答幾個問題（選「Voice」和「With code」）
  5. 完成註冊，進入 Console

取得 Account SID 和 Auth Token

在 Twilio Console 首頁就能看到兩個重要的值：

Account SID：AC1234567890abcdef...
Auth Token：（點擊眼睛圖示顯示）
⚠️ Auth Token 是密碼等級的敏感資訊，不要分享給任何人，不要放到 GitHub 上。

購買電話號碼

  1. 在 Console 左側選單點 Phone Numbers → Manage → Buy a number
  2. 選擇國家（建議選美國 United States，最便宜）
  3. 勾選「Voice」能力
  4. 選一個你喜歡的號碼，按 Buy
  5. 試用帳號免費買一個號碼

20.3 設定龍蝦接聽語音通話

安裝 Twilio 外掛

openclaw plugins install twilio-voice

設定認證資訊（TOOLS.md）

## Twilio（語音通話）
- Account SID: 你的_Account_SID
- Auth Token: 你的_Auth_Token
- 電話號碼: +1xxxxxxxxxx（你買的號碼）
- 語音語言: zh-TW（中文）
- TTS 引擎: google（Google 語音合成）

設定 Webhook

  1. 進入 Phone Numbers → Manage → Active numbers
  2. 點擊你的號碼
  3. 在 Voice Configuration 區塊：
    • A call comes in → Webhook
    • URL 填入：https://lobster.你的域名.com/twilio/voice
    • 方法選 POST
  4. 按 Save configuration
openclaw gateway restart    # 重啟 Gateway

20.4 語音通話的運作原理

通話流程

1. 你撥打 Twilio 號碼
2. Twilio 接聽，通知 OpenClaw（透過 Webhook）
3. OpenClaw 送出歡迎語（TTS 語音合成 → 播放給你聽）
4. 你開始說話
5. Twilio 即時語音辨識（STT），把你的話轉成文字
6. OpenClaw 把文字送給 AI 模型處理
7. AI 產生回覆文字
8. TTS 把回覆轉成語音，播放給你聽
9. 重複 4-8 直到掛斷
💡 整個過程的延遲大約 1-3 秒——你說完話後，龍蝦需要一兩秒才會回答。比打字聊天稍微慢一點，但完全可以接受。

20.5 實際應用場景

🎙️ 個人語音助手

打電話問行程、改行程、查資料。「幫我查一下明天下午有什麼行程？」「把四點半的改到五點。」

👴 長輩友善介面

阿公打電話問天氣：「明天要不要帶傘？」龍蝦：「明天台中 22 度，有小雨喔，出門記得帶傘。」

🏪 語音客服機器人

客戶打來問營業時間、訂位。龍蝦自動回答、幫忙預約，不用你親自接電話。

⏰ 電話提醒

搭配 HEARTBEAT.md，龍蝦每天早上八點打電話報行程和天氣：「早安！今天有一個行程：下午 2 點看牙醫。」

20.6 進階設定與常見問題

進階設定

🎙️ 通話錄音

自動錄音方便回顧，設定保留天數，儲存到 workspace/data/recordings/

🚫 來電黑名單

避免騷擾電話。可設黑名單或白名單模式（只接白名單電話）

⏱️ 通話時長限制

避免意外高額費用。最長 10 分鐘、每日上限 30 分鐘

常見問題

問題回答
台灣手機可以打美國號碼嗎？可以，但有國際通話費。建議用 LINE Out 或其他 VoIP 撥打更便宜
語音辨識聽不懂台灣國語？設定 zh-TW，說慢一點、安靜環境，或升級 Google STT
通話中可以操作電腦嗎？可以！搭配 Desktop Agent（CH18），語音指令操作電腦
可以多人同時打嗎？可以，每通電話獨立對話、獨立計費
有其他選擇嗎？目前官方支援 Twilio，社群開發中 SIP/WebRTC 方案
